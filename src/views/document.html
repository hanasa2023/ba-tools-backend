Ba Tools Api

欢迎使用Ba Tools API

此项目使用 Schale DB 作为数据源，并在此基础上新增了学生的live 2d图像

✨ 使用方法

  • 此 api 的 url 路径与 Schale DB 的路径基本相同
    如果你想访问 Schale DB 中 data/zh/students.json 的数据，那么你可以通过 GET https://api.hanasaki.tech/data/zh/students.json 来访问该数据
  • 新增 live 2D image 的请求路径为 images/student/l2d/<学生id>.webp
    例如： https://api.hanasaki.tech/images/student/l2d/10000.webp
